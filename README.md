# 🎵 AI Music Prompter

## 中文

### 什么是 AI Music Prompter？

一个专注于 AI 音乐生成的 Claude Code Skill。覆盖 **8 个维度** 的提示词工程（人声、风格、情绪、节奏、乐器、质感、年代、混音），以及为歌词设计曲式结构——Intro、Verse、Chorus、Bridge、Outro，每段配有能量标记和人声方向。

### 安装

在 Claude Code 中运行：

```
/plugin marketplace add dukeko369/ai-music-prompter
```

### 八个维度

| 维度 | 内容 |
|------|------|
| 🎤 人声类型 | 性别、音域、音色、唱腔（含京/昆/秦/越剧） |
| 🎵 音乐风格 | 300+ 标签，City Pop → Amapiano → 古风 |
| 🎭 情绪氛围 | 20 种情绪色彩 |
| 🥁 节奏速度 | BPM 范围 + 节奏型 + 拍号 |
| 🎸 乐器配置 | 40+ 乐器，含中国民乐 |
| 🔊 声音质感 | 20 种音色纹理 |
| 📻 年代风格 | 60s → Y2K → 未来感 |
| 🎧 混音后期 | 混响/延迟/调制/空间/母带 |

### 曲式结构

8 套风格模板：流行 · EDM · 嘻哈 · 金属 · 电影配乐 · 古风 · 爵士 · 短小。过渡设计 + 尾声风格 + 能量曲线可视化。

### 平台支持

- **Suno** — 完整标签体系（Energy/Mood/Structure/Transition/Callback）
- **Udio** — 通用格式 + 30 秒分段策略
- **Lyria** — 全句提示词 + 多语言
- **Eleven Music** — 商业级人声

### 用法

说「帮我在 Suno 上做一首 City Pop，女生唱，夏天开车听」，Skill 会自动追问细节，输出可复制的提示词和歌词结构。

## English

### What is AI Music Prompter?

A Claude Code skill for AI music generation. It covers **8 dimensions** of prompt engineering — vocal type, genre, mood, tempo, instruments, texture, era, and mix — plus lyric-to-song-structure design with Intro, Verse, Chorus, Bridge, and Outro, each annotated with energy levels and vocal directions.

### Install

Run in Claude Code:

```
/plugin marketplace add dukeko369/ai-music-prompter
```

### Eight Dimensions

| Dimension | Coverage |
|-----------|----------|
| 🎤 Vocal | Gender, range, timbre, delivery (incl. Chinese opera) |
| 🎵 Genre | 300+ tags from City Pop to Amapiano |
| 🎭 Mood | 20 emotional colors |
| 🥁 Tempo | BPM ranges + time signatures + groove feel |
| 🎸 Instruments | 40+ instruments including Chinese traditional |
| 🔊 Texture | 20 timbral textures |
| 📻 Era | 60s → Y2K → futuristic |
| 🎧 Mix | Reverb, delay, modulation, spatial, mastering |

### Song Structure

8 genre-specific templates: Pop · EDM · Hip Hop · Metal · Cinematic · Chinese Ancient · Jazz · Compact. Transition design + outro styles + energy curve visualization.

### Platforms

- **Suno** — Full tag vocabulary (Energy/Mood/Structure/Transition/Callback)
- **Udio** — Universal format + 30s segment strategy
- **Lyria** — Full-sentence prompts + multilingual
- **Eleven Music** — Commercial-grade vocals

### Usage

Say "make me a City Pop track on Suno, female vocal, summer driving vibe" — the skill interviews you, then outputs a copy-paste-ready prompt and annotated lyric structure.

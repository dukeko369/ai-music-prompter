# 🎵 AI Music Prompter

AI 音乐提示词工程助手 — 从 8 个维度生成专业 AI 音乐提示词，支持歌词曲式结构设计。

## 安装

在 Claude Code 中运行：

```
/plugin marketplace add dukeko369/ai-music-prompter
```

## 功能

- 🎤 **人声类型** — 性别、音域、音色、唱腔（含中国戏曲）
- 🎵 **音乐风格** — 300+ 标签，从 City Pop 到 Amapiano
- 🎭 **情绪氛围** — 20 种情绪色彩
- 🥁 **节奏速度** — BPM 范围 + 节奏型
- 🎸 **乐器配置** — 40+ 乐器选择
- 🔊 **声音质感** — 20 种音色纹理
- 📻 **年代风格** — 60s 到未来感
- 🎧 **混音后期** — 混响/延迟/调制/空间效果

### 曲式结构设计

8 套风格模板（流行/EDM/嘻哈/金属/电影/古风/爵士/短小）+ 过渡设计 + 尾声风格 + 能量曲线可视化。

## 平台支持

- **Suno** — 完整标签体系（Energy/Mood/Structure/Transition tags）
- **Udio** — 通用格式 + 30秒分段策略
- **Lyria** — 全句风格提示 + 多语言支持
- **Eleven Music** — 商业级人声描述

## 示例

说"帮我在 Suno 上做一首 City Pop 风格的歌，女生唱的，夏天开车听"，Skill 会自动追问细节，然后输出可直接复制粘贴的完整提示词和曲式结构。
